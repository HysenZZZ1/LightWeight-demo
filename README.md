# LightWeight-demo
深度学习轻量化模型学习仓库，记录从经典CNN到轻量化结构的实践与思考。

## 已完成项目
- **Depthwise 卷积实现 + CIFAR-10 实验**
  实现深度可分离卷积结构，在CIFAR-10上进行训练，理解轻量化卷积的计算逻辑与结构特性。
- **VGG 网络实现 + CIFAR-10 图像分类**
  复现经典VGG网络，完成CIFAR-10图像分类任务，掌握CNN模型搭建、训练与调优流程。
