# 01-Depthwise-CNN-CIFAR10 实验记录

## 1. 实验目标
验证深度可分离卷积在CIFAR-10小尺寸图像分类任务上的效果，对比其与标准卷积的差异。

## 2. 关键实现与修改
- 实现了深度可分离卷积（Depthwise + Pointwise）模块
- 调整下采样策略：延后使用 `stride=2`，改为先同尺度堆叠卷积提取特征，再下采样，减少信息过早丢失

## 3. 实验结果
- 验证集准确率：75%
- 现象：训练集与验证集精度差距很小，模型表现为明显欠拟合

## 4. 结论与反思
- 深度可分离卷积对32×32小图任务并不友好：
  1.  底层通道信息交互不足，特征提取能力弱
  2.  小图本身有效特征稀缺，拆分卷积会加剧信息损耗
- 小数据集、小分辨率图像分类任务，优先选择标准卷积（VGG风格）更合适
- 轻量化结构更适合大图、大数据集与端侧部署场景

##5. 本次实验小小的知识补充：
  1. RandomHorizontalFlip 水平翻转：语义完全不变，猫还是猫、车还是车，属于温和无伤害增强，几乎所有模型都能吃。
  2. RandomCrop 随机裁剪 + padding：会切掉边缘、偏移主体，图片局部结构
变多、有效特征被打乱，属于高难度增强。
     结论：模型小，训练次数少的情况下，不适合用crop,反而会掉精度。
